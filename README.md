## To Run 

On command line, run:

```
$ python processing.py [location of part1 data file - try data/*questionnaire-l2e5.csv] [location of part2 data file - try data/*task-gh5n.csv]
```

Output files will include (labeled with participant unique ID):

1. List of all relevent values, including reaction time, condition, and if response was correct.
2. Accuracy means for conditions.
3. Precision means for conditions.
